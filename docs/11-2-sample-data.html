11.2 Sample Data

library(ROCR)

This chapter will use a datasets that contain 12,000 recodds. Here are the variables:

Dependent Variable

  • left: employee had left the company

Independent Variables

  • S: Satisfied Employee (through survey)
  • NP: Number of Project Currently Handled
  • LPE: Last Project Evaluation
  • ANH: Average Number of Monthly Hours
  • TIC: Time in company (in years)
  • Newborn: Had a new born in last 12 months
