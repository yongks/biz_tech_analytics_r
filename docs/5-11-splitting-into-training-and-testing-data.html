5.11 Splitting Into Training and Testing Data

When we have only ONE dataset, we can split them into training and testing, example 0.7/0.3 split.

sample.split takes a vector (or a column from a data.frame), randomly assign TRUE (training data) and FALSE (test data) according to a specified ratio. It returns a logical vector for data subsetting purpose.

sample.split (Y, SplitRatio = 2/3, group = NULL)
$\quad$ Y = vector
$\quad$ SplitRatio = ratio of split

Note sample.split doesn’t take a data.frame, it takes a vector (or a data.frame column).

Load the library, display the original data.frame. Split the data and verify the logical vector output.

set.seed(8034)
my.df
##     id value1 value2
## 1 ID_1     55     36
## 2 ID_2     81     10
## 3 ID_3     79     19
## 4 ID_4     98     35
## 5 ID_5     90     29
split.data = sample.split(my.df$id, 0.6)  # randomly 0.6 set to TRUE, remaining FALSE
split.data
## [1] FALSE FALSE  TRUE  TRUE  TRUE

Subseting for the training data

subset( my.df, split.data )   # Training Data (0.6)
##     id value1 value2
## 3 ID_3     79     19
## 4 ID_4     98     35
## 5 ID_5     90     29
#my.df[  split.data, ]        #   alternative

Subsetting the test data by negating the splitting logical vector.

subset( my.df, !split.data )  # Testing Data (0.4)
##     id value1 value2
## 1 ID_1     55     36
## 2 ID_2     81     10
#my.df[ !split.data, ]        #   alternative
