6.3 Finding Data

Find the first match row number(s) of specific element(s).

match ( x, y )
\(\quad\) x = vector of criteria
\(\quad\) y = vector of elements to look in

match( 'D2', my.df$dept )
## [1] NA
match( c('D2','D3'), my.df$dept )
## [1] NA NA
