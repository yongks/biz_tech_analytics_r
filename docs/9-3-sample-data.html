9.3 Sample Data

Sample data used in this chapter emulate two dimensional data points with three groups with clear grouping when visualize.

set.seed(1234)
my.df = data.frame(
  id = paste('ID_', 1:12, sep = ''),
  grp = c(rep('G1', 4), rep('G2', 4), rep('G3', 4)),
  value1 = c( round(rnorm(4, mean = 10,  sd = 3)),
              round(rnorm(4, mean = 10, sd = 3)),
              round(rnorm(4, mean = 30, sd = 3))),
  value2 = c( round(rnorm(4, mean = 10, sd = 3)),
              round(rnorm(4, mean = 20, sd = 3)),
              round(rnorm(4, mean = 20, sd = 3))),
  stringsAsFactors = F
)
rownames(my.df) = my.df[,1]
my.df
